## 🛠️ How to Restore (Apne System ko Restore Kaise Karein)
If the main trading engine files in the parent folder ever get corrupted, broken, or misconfigured:

### Option 1: Double-Click the Restore Script (Recommended)
1. Double-click the `restore.bat` file located in this folder.
2. It will ask for confirmation. Type `Y` and press **Enter**.
3. It will automatically overwrite the parent directory's files with this stable version.

### Option 2: Manual Restore
1. Copy all the files from this `Trilochan 1.0` directory.
2. Paste them into the parent directory, replacing the existing files.
